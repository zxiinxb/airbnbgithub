🏙️ The NYC Airbnb Story – Data Analysis Project
1️⃣ Project Overview
This project analyzes the NYC Airbnb 2019 dataset to understand patterns in listings, pricing, and popularity across New York City. The analysis focuses on how factors like **location, room type, and reviews** influence Airbnb listings and guest preferences.

2️⃣ Dataset
The dataset contains 48,000+ Airbnb listings across 5 boroughs of New York City:

- Manhattan  
- Brooklyn  
- Queens  
- Bronx  
- Staten Island  

Key Features in the Dataset
- 💰 Price per night  
- 🏠 Room type  
- 📍 Neighborhood and borough  
- ⭐ Number of reviews  
- 📅 Availability (365 days)  
- ⏳ Minimum nights required  
- 👤 Host information

5️⃣ Key Insights 📈

Some important findings from the dataset:

- 🏠 52% listings are Entire Homes
- 👥 45% listings are Private Rooms
- 🛏️ Only 3% listings are Shared Rooms

- 💰 Average nightly price: ~$150
- 💎 Manhattan average price: ~$212

- ⭐ Average 1.4 monthly reviews per listing
- 📍 Manhattan and Brooklyn together host the majority of listings.

- 🏡 A highly popular listing with many reviews was found in **Queens**, showing that value and hospitality can compete with location.
